## About the Project

Seshly is developed by **AutoXyrium** as a next-generation academic platform.  
The app connects students across different campuses and provides powerful tools such as real-time tutoring, AI-guided learning sessions, shared notebooks, and a marketplace for study materials.

This repo contains the codebase for the **Seshly mobile application**, built with Flutter and optimized for Android and iOS.


## Key Features

### **1. AI-Powered Sesh Screen (Intelligent Notebook)**
- Write, store, and organize notes
- Improve and optimize notes using AI
- Convert rough drawings into clean, labeled diagrams
- Auto-generate practice questions based on topic
- Shared live notebook for tutoring sessions
- Auto-save diagrams, explanations, and content from live sessions


### **2. Academic Question Feed**
- Post academic questions
- Receive help from peers across campuses
- AI helper assists with step-by-step solutions
- Upvote high-quality answers


### **3. One-Tap Tutor Marketplace**
- Find tutors instantly (like Upwork for academics)
- Tutors set hourly rates
- Book 1-on-1 sessions
- Rich tools for math, science, and technical subjects during video calls


### **4. Notes & Books Marketplace**
- Buy and sell notes
- Trade textbooks and study supplies
- Ratings and reviews for credibility


### **5. Messaging & Real-Time Communication**
- 1-on-1 chat
- Group chats
- High-quality video calls
- Shared whiteboard features


### **6. Calendar & Reminders**
- Syncs with university schedules
- Auto-imports classes, tests, and assignments
- Smart reminders and study planning


### **7. Mentorship Program**
- Connect with senior or postgraduate mentors
- Guided academic and career support


### **8. Smart UI/UX**
- Clean academic theme with dark mode by default  
- Designed for productivity and fast navigation  
- Student number login for verified campus identity  


## Tech Stack

- **Frontend:** Flutter  
- **Backend:** Firebase  
- **Platforms:** Android, iOS (with supp for macOS, Web, Linux & Windows coming later)  
- **AI:** Integrated structured reasoning APIs for step-by-step guidance and diagram interpretation
